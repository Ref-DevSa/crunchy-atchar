# Crunchy Atchar Website Redesign Brief

## 1. Company

**Name:** Crunchy Atchar

**Industry:** Food Manufacturing / Homemade Atchar

**Website:**
https://crunchy-atchar.co.za/

---

## 2. How I found the company

I found Crunchy Atchar through an online search while looking for South African food businesses that could benefit from a modern website redesign. Their existing website provided a good opportunity to practise applying React and Tailwind CSS to a real business.

---

## 3. Why I chose this company

I chose Crunchy Atchar because it is a real South African business with genuine products and branding. Their website already contains useful information about the company and its products, making it a realistic redesign project rather than creating a fictional website.

The project also allowed me to practise building reusable React components, responsive layouts, routing, and modern user interface design while keeping the company's identity.

---

## 4. Why the website needs a redesign

After reviewing the current website, I identified several areas that could be improved.

### Visual Design

The website has a basic layout that does not immediately communicate a premium food brand.

### Navigation

The navigation could be clearer and make it easier for visitors to move between important pages.

### User Experience

The homepage could better showcase the products using larger product images and stronger calls-to-action.

### Product Presentation

Products could be displayed using modern product cards with consistent spacing, pricing and descriptions.

### Mobile Responsiveness

Although usable, the mobile experience can be improved through better spacing and responsive layouts.

### Calls To Action

Buttons encouraging users to order products or contact the business could be more prominent.

### Branding

The redesign uses consistent colours, typography and imagery that better match the company's identity and products.

---

## Improvements made

- Built with React and Tailwind CSS
- Responsive design
- Modern navigation
- Hero section with clear call-to-action
- Product catalogue
- Product search
- Category filtering
- About page
- Contact page
- Testimonials
- Reusable React components
- Improved typography
- Improved spacing
- Better visual hierarchy

This redesign demonstrates how the company's online presence could be modernised while maintaining its existing branding.
